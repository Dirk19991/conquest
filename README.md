Сделаны адаптивы для планшета и мобильного устройства (на свое усмотрение). Использован сборщик parcel. Инструкция по запуску:

```
git clone https://github.com/Dirk19991/conquest.git
git cd .\conquest\
npm install
npm start
```

Сборка: 

```
git clone https://github.com/Dirk19991/conquest.git
git cd .\conquest\
npm install
npm run build
```
